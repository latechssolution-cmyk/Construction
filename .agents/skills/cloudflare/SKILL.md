# Cloudflare Edge Infrastructure

Cloudflare allows deploying serverless edge Workers, D1 SQL databases, R2 object storage, and messaging Queues.

## Usage
- Deploy serverless workers using Wrangler CLI.
- Provision storage (R2 buckets) or relational databases (D1).
