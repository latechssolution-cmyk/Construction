# Firecrawl Web Reader

Firecrawl crawls website URLs and outputs clean, LLM-optimized markdown representations, executing Client-side JavaScript.

## Usage
- Scraping complex, client-side rendered docs, single-page apps, or blogs.
- Fetch raw text content from URLs for semantic indexing or summaries.
