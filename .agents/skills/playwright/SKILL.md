# Playwright Browser Automation

Playwright allows headless or headful browser automation to control, test, or scrape pages where no direct API exists.

## Usage
- Drive actions on pages (clicking, filling forms, taking screenshots).
- Automate end-to-end user flows for verification and regression testing.
